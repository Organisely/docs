`@organisely/agent-sdk` is the official TypeScript SDK for building agents on Organisely. It exposes every platform capability — connectors, memory, events, approvals, activity tracking — through a single instance-based client.

## Installation

<CodeGroup>

```bash npm
npm install @organisely/agent-sdk
```

```bash pnpm
pnpm add @organisely/agent-sdk
```

```bash yarn
yarn add @organisely/agent-sdk
```

```bash bun
bun add @organisely/agent-sdk
```

</CodeGroup>

## Initialize the client

```ts
import { OrganiselyAgent } from '@organisely/agent-sdk'

const client = new OrganiselyAgent({
  baseUrl: 'https://api.useorganisely.com',
  apiKey: process.env.ORGANISELY_API_KEY,
})
```

`OrganiselyAgent` is an alias for `Organisely`. Both are identical.

## Authentication options

The client accepts one of three auth shapes:

```ts
// API key
new OrganiselyAgent({ apiKey: 'key_...' })

// Short-lived access token
new OrganiselyAgent({ accessToken: 'eyJ...' })

// Token provider (OAuth rotation)
new OrganiselyAgent({
  tokens: {
    async get() { return { accessToken: await store.read() } },
    async set(tokens) { await store.write(tokens) },
  }
})
```
