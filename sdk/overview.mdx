`@organisely/agent-sdk` is the official SDK for building agents on Organisely.

## What this SDK includes

<CardGroup cols={2}>
  <Card title="Agent definition" icon="robot">
    Define AI agents with model, provider, and event handlers.
  </Card>
  <Card title="Event subscriptions" icon="bell">
    Subscribe to provider events and poll for delivery.
  </Card>
  <Card title="Connector catalog" icon="plug">
    List connector definitions and available capabilities.
  </Card>
  <Card title="Connection management" icon="link">
    Connect tenant-scoped runtime instances.
  </Card>
  <Card title="Tool discovery" icon="search">
    Discover MCP tools and call them through Organisely's proxy.
  </Card>
  <Card title="Memory" icon="database">
    Key-value session storage for agent state.
  </Card>
</CardGroup>

## Minimal example

```ts
import { OrganiselyAgent } from '@organisely/agent-sdk';

const client = new OrganiselyAgent({
  baseUrl: 'https://api.useorganisely.com',
  apiKey: process.env.ORGANISELY_API_KEY,
});

const agent = client.define('email-drafter', {
  model: 'gpt-4.1',
  provider: 'openai',
  async handler(event) {
    const draft = await draftReply(event.payload.subject);
    await client.memory.set('last_draft', draft.id);
  },
});

await agent.subscribe(['email.received']);
const controller = new AbortController();
await agent.run({ signal: controller.signal });
```

## Connectors and connections

Organisely acts as an **MCP execution gateway**. You connect a connector for a tenant, Organisely stores credentials, manages OAuth, and proxies all MCP tool calls.

```ts
// Browse available connector definitions
const { connectors } = await client.connectors.list()

// Connect via hosted OAuth
const { connectorId, authUrl } = await client.connections.connect(tenantId, 'slack')

// Or with a bearer token
const { connectorId } = await client.connections.connectWithToken(tenantId, 'github', {
  type: 'bearer',
  token: 'ghp_...',
})

// Or a custom MCP server URL
const { connectorId } = await client.connections.connectUrl(tenantId, {
  url: 'https://my-custom-mcp.example.com',
})
```

## Calling tools

```ts
// Discover tools
const tools = await client.connections.tools(tenantId, connectorId)

// Call a tool (Organisely proxies the call)
const result = await client.connections.call(tenantId, connectorId, 'send_message', {
  channel: '#general',
  text: 'Hello!',
})
```
