title: Introduction
description: Organisely is a team-shared agentic operating layer — runtime, orchestration, memory, policy, and connectors.
---

Organisely provides three layers for building and running agents in a team context:

1. **Team apps and operators** — the products and people using the platform
2. **Organisely SDK and API** — the code surface for connecting agents to the platform
3. **Shared operating layer** — execution, approvals, memory, connectors, and audit trails
